# 🪑 Furniture Builder - 커스텀 가구 제작 프로그램

> 디자인 패턴을 활용하여 사용자가 원하는 색상과 옵션으로 가구를 제작하는 Python GUI 프로그램

<br>

## 📋 프로젝트 정보

| 항목 | 내용 |
|------|------|
| 개발 기간 | 2024.03 ~ 2024.06 |
| 팀 구성 | 3인 팀프로젝트 |
| 담당 역할 | 전체 코드 작성, 이미지 구현 및 확인 |
| 과목 | 설계패턴 |

<br>

## 📌 프로젝트 소개

간단하고 직관적인 GUI를 통해 사용자가 원하는 색상과 옵션으로 가구를 제작할 수 있는 프로그램입니다.
의자, 옷장, 책상 3가지 가구 유형을 지원하며, 4가지 디자인 패턴을 적용하여 효율적인 구조로 설계했습니다.

<br>

## 🛠 기술 스택

| 분야 | 기술 |
|------|------|
| 언어 | Python |
| GUI | Tkinter (ttk, OptionMenu, Checkbutton) |
| 이미지 처리 | Pillow (PIL) |

<br>

## 👤 내가 맡은 역할

- **전체 코드 설계 및 구현** (`furniture_builder.py`)
- 4가지 디자인 패턴 적용 및 통합
- 가구 유형별 이미지 파일 구성 및 화면 표시 구현
- GUI 레이아웃 설계 및 인터랙션 구현

<br>

## ✨ 주요 기능

- **가구 생성**: 가구 유형(의자/옷장/책상), 색상, 추가 옵션 선택 후 생성
- **옵션 선택**: 가구별 추가 옵션 제공
  - 의자: 바퀴(Wheel), 팔걸이(Armrest)
  - 옷장: 거울 문(Mirror Door)
  - 책상: 높이 조절(Adjustable Height)
- **이미지 표시**: 선택한 옵션에 맞는 가구 이미지 화면에 표시
- **되돌리기(Undo)**: 이전에 생성된 가구 상태로 복원

<br>

## 🎨 가구 색상 옵션

| 가구 | 선택 가능한 색상 |
|------|----------------|
| 의자 | White, Grey, Green |
| 책상 | Black, White, Wood, Walnut |
| 옷장 | Darkgrey, Grey, White, Wood |

<br>

## 🏗 적용된 디자인 패턴

| 패턴 | 클래스 | 역할 |
|------|--------|------|
| **Builder** | `FurnitureBuilder` | 가구 객체를 단계별로 생성 |
| **Decorator** | `FeatureDecorator` 및 하위 클래스 | 가구에 옵션(바퀴, 팔걸이 등)을 동적으로 추가 |
| **Flyweight** | `FlyweightFactory`, `Color` | 색상 객체를 공유하여 메모리 사용 최적화 |
| **Memento** | `Memento`, `Caretaker` | 가구 상태 저장 및 되돌리기(Undo) 기능 구현 |

## 💡 트러블슈팅 & 배운 점

- **패턴 선택의 어려움**: 다양한 디자인 패턴 중 프로젝트에 적합한 패턴을 선정하는 과정에서 각 패턴의 개념을 직접 분석하고 팀원과 논의하며 결정
- **코드 통합 문제**: 서로 다른 패턴들을 하나의 코드로 합치는 과정에서 충돌 발생 → 코드 리뷰와 지속적인 리팩토링으로 해결
- 디자인 패턴을 실제 프로젝트에 적용하면서 코드의 **재사용성, 확장성, 가독성**이 향상됨을 직접 경험
